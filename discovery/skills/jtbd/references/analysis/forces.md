# Forces of Progress — Template and Analysis

## The Framework

The decision to adopt (or abandon) a solution is determined by the balance of 4 forces:

```
[Push]                              [Pull]
Frustration with current state  ←→  Attraction to the new solution
        ↑                                    ↑
        |_____________ DECISION ____________|
        ↓                                    ↓
[Habit]                             [Anxiety]
Comfort with current behavior       Fear of change / uncertainty
```

**Key insight:** Reducing Anxiety and Habit is often more effective than increasing Pull.
Many products fail not because they are bad, but because they do not remove the resistance forces.

```
**Recommendation:**

- If Push is high but adoption is low: verify whether Pull and Anxiety reduction are addressed
- If Push and Pull are high, Anxiety and Habit are low: the problem is likely awareness or access
```

## How to Integrate with PR/FAQ

In the "Why now?" section of the PR/FAQ:
- High Push → real urgency documented with evidence
- Existing Pull → confirmed latent demand
- Identified Anxiety → the product must address this explicitly in the value proposition
- Habit → onboarding and adoption strategy needs attention
