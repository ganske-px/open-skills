# Story Type: Data Insight (SWD)

## When to use
- The data IS the story — the visual carries the message
- Product analysis, metrics review, diagnosis
- "Look at what the data shows about X"
- Report for squad, board, or stakeholder focused on evidence

## Framework: Storytelling with Data (Knaflic)

Data needs context, finding, and implication to become a story. A number without context is noise.

### Required structure

```
CONTEXT   → why this data matters now
FINDING   → what the data shows (title = the message, not the description)
SO WHAT   → implication for the business or product
NEXT STEP → what we do with this
```

## Chart type identification

| Data | Type |
|---|---|
| Comparison between categories | `bar_vertical` (≤6 labels) or `bar_horizontal` (>6 or long labels) |
| Time trend | `line` |
| Two points in time | `slope` |
| Relationship between variables | `scatter` |
| Few categories with precision | `dot_plot` |
| Output from probability-analysis | `confidence_interval` |

## Skills to invoke

| When | Skill |
|---|---|
| Data not available | `data-research` — collect before any visual |
| Comparisons, rates, probabilities | `probability-analysis` — always before comparing groups |
| CI calculated | use `confidence_interval` chart_type mandatorily |
| Implication involves prioritization | `product-decision` |

## Non-negotiable SWD rules

| Principle | Rule |
|---|---|
| Message in the title | "Approvals fell 18%", not "Approval Rate by Month" |
| One highlight color | `#E8533F` for the main element; `#C8C8C8` for everything else |
| Zero clutter | Remove gridlines, borders, legends when direct labels solve it |
| Data in Appendix | Query/source always present — non-negotiable |
| No pie | Never pie chart |
| No dual Y-axis | Two separate charts if necessary |
| Max 5 charts per HTML | If more, question whether it's a story or a dashboard |

## Contextual chart rules

| Decision | Criterion |
|---|---|
| Vertical vs horizontal bar | Horizontal when labels > 20 chars or categories > 6 |
| Line vs bar | Line for continuous time; bar when individual point matters |
| Slope chart | Only two time points being compared |
| Annotation vs legend | Inline when ≤ 3 series |
| Show CI | Always when data comes from probability-analysis |

## Recommended output

| Audience | Preferred format |
|---|---|
| Any | `html-story` with TL;DR at the top + Appendix with queries |
| Quick async share | `png` generated by chart_builder |
